Activities — 活動内容

3つの領域で、
ITに向き合う。

Bashi.com の活動は、大きく3つの領域に分かれています。それぞれが独立しているのではなく、ひとつのプロジェクトの中で重なり合いながら、つくる・解く・表現するを行き来します。

01 / 03

技術・プロダクト開発

アイデアを「実際に動くもの」にするのが、この領域の役割です。Web・モバイル・AI まで、つくりたいものに合わせて技術を選び、企画からリリースまでを経験できます。

Web Mobile AI / ML Backend Infra

Web・モバイルアプリ

フロントからバックエンドまで、チームで一つのプロダクトを形にします。

AI・データ活用

機械学習やデータ分析を使った機能づくりに挑戦します。

個人開発のシェア

それぞれが作ったものを持ち寄り、フィードバックし合います。

02 / 03

ITを用いた課題解決

身近にある「不便」や「面倒」を、テクノロジーで仕組みごと変えていく領域です。学内の運用やイベントなど、リアルな課題を題材に取り組みます。

業務改善 自動化 データ 効率化

学内の課題に挑む

日々の運用や手作業を、ツールや自動化で軽くします。

業務効率化・自動化

スクリプトや小さなツールで、繰り返し作業をなくします。

仮説 → 検証

小さくつくって試し、本当に役立つかを確かめます。

03 / 03

デジタルクリエイション

表現としての IT を楽しむ領域です。デザイン・映像・ゲームなど、技術と創造性が交わるものづくりに、自由に取り組めます。

UI/UX デザイン 映像 Game

UI / UX デザイン

使いやすさと美しさの両立を、手を動かしながら学びます。

映像・モーション

動きのある表現で、伝わるコンテンツをつくります。

ゲーム・体験制作

インタラクティブな作品づくりに挑戦します。
